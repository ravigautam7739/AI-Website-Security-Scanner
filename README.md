# 🛡️ JARVIS Website Security Scanner

A Python-based **website security analysis simulator** that performs a visual security scan process and generates a website safety report.

This project creates a **terminal-based cybersecurity scanner interface** inspired by AI assistants like JARVIS.

---

# 🚀 Features

✔ Website URL input
✔ Animated security scanning process
✔ Multiple security check simulation
✔ Threat level generation
✔ Security score calculation
✔ Cybersecurity-style terminal UI

# 🧠 How It Works

1. User enters a website URL

Example:

```text
Enter Website URL:
https://example.com
```

2. Scanner starts performing checks:

```text
Checking SSL Certificate
Inspecting HTTP Headers
Scanning For Malware
Checking DNS Records
Analyzing Source Code
Checking Phishing Risk
```

3. After scanning:

The system generates:

* Threat Level %
* Security Score
* Website Status
